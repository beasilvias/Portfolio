IFE LEVEL

Redesigning an In-Flight Entertainment interface

Redesign of the In-Flight Entertainment System of the spanish airline LEVEL by Iberia. I followed the basic patterns they stablished with their brand and added some features that could be useful. Developed a minimal interface and changed their white theme for a darker theme, so that is more visually comfortable for the user inside the mostly dark atmosphere of the plane.

ROLE   Sole UI Designer

CONTEXT   User Interface Design of LEVEL airline
